### 📂 프로젝트 목표
- OTT 시장과 숏폼 시장의 성장 추이 및 현재 대중의 인식 조사
- 음성과 텍스트 데이터를 활용한 감정 분석 모델 제작
- 감정 분석 결과를 기반으로 특정 감정에 맞는 영상 클립을 제공하는 홈페이지 제작
<br></br>
